# MC Asisten AI

A backend Node.js project that creates a virtual assistant powered by AI using the OpenAI API and Express.js.

---

## Features

- Provides an API endpoint to ask questions and receive AI-generated answers.
- Supports multiple languages for emergency contact information.
- Environment variable support for secure API keys and configuration.
- Simple, lightweight Express server setup.

---

## Requirements

- Node.js (version 16 or higher recommended)
- npm (Node package manager)
- OpenAI API key (You need to sign up at [OpenAI](https://platform.openai.com/) to get an API key)
